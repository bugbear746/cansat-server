# CanSat Flask Server (Render + UptimeRobot Ready)

## Features
- Live video stream from Pi Camera or USB Camera
- Real-time telemetry graphs for altitude and temperature
- Auto-deployable on Render.com
- Free server uptime using UptimeRobot

## Deployment
1. Push this project to GitHub
2. Deploy to Render using Docker environment
3. Add a monitor on UptimeRobot with the Render URL

## UptimeRobot URL format
https://your-subdomain.onrender.com
